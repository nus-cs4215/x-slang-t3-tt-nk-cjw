## User Guide

We currently provide a non-interactive repl. Run `yarn repl` to see USAGE.

Look at `/examples` for sample `*.rkt` source files. You can run these:

``` sh
yarn repl -v examples/basic.rkt
yarn repl examples/kanren.rkt
```
